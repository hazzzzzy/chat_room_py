### 亮点

- 关于同浏览器多窗口，在redis中存储用户的连接信息，连接时验证，如果存在则断开之前的连接，重新连接
- 关于头像，使用的是对象存储公有读，但是使用hmac加密userid来命名图片，例如：{存储桶链接}/{jwt加密的用户id}.png|jpg|jpeg，防止轮询用户id爬取头像
